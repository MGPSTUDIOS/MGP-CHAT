## 📥 Как установить приложение

1. **Скачайте файл**  
   Нажмите на кнопку **«Download»** рядом с файлом `app-release.apk` в разделе(укажем потом)
2. **Разрешите установку из неизвестных источников**  
   При открытии файла Google может предупредить:  
   > *«Это приложение не прошло проверку Play Protect. Установка может быть опасной»*  
   Нажмите **«Продолжить»** → **«Установить»** → **«Разрешить установку из неизвестных источников»** (настройки телефона → Безопасность).

3. **Завершите установку**  
   После загрузки откройте файл `app-release.apk` и нажмите **«Установить»**.
